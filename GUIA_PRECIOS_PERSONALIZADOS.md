## 1️⃣ CREAR PRODUCTO CON PRECIOS PERSONALIZADOS

### Formulario de Producto:
```
┌──────────────────────────────────────────────────┐
│ INFORMACIÓN BÁSICA                               │
│ Nombre: [Paracetamol 500mg_______________]       │
│ Categoría: [Analgésicos ▼]                       │
│ Precio Compra: [0.50___]                         │
│ Precio Venta (Unidad): [1.00___]  ← Precio base │
│                                                  │
│ PRECIOS POR PRESENTACIÓN (OPCIONALES)            │
│ Precio por Blister (10 un.): [9.00___]          │
│ Precio por Caja (100 un.):   [75.00___]         │
│                                                 │
│ 💰 Preview de Precios de Venta:                 │
│ ┌─────────────────────────────────────────────┐ │
│ │ Por Unidad    │ Por Blister   │ Por Caja   │ │
│ │ S/. 1.00      │ S/. 9.00      │ S/. 75.00  │ │
│ │ Precio base   │ 10.0% desc.   │ 25.0% desc.│ │
│ └─────────────────────────────────────────────┘ │
│                                                  │
│ CONFIGURACIÓN PRESENTACIÓN                       │
│ Unidades por Blister: [10___]                    │
│ Blisters por Caja:    [10___]                    │
│                                                  │
│          [Cancelar]  [Guardar Producto]          │
└──────────────────────────────────────────────────┘
```

### 💡 Tips:
- Si dejas los precios vacíos → Sistema calcula automáticamente (10% y 20% desc.)
- El % de descuento se calcula automáticamente según TUS precios
- Puedes dar descuentos diferentes (5%, 15%, 30%, etc.)

## 2️⃣ HACER UNA VENTA

### Módulo de Nueva Venta:
```
┌──────────────────────────────────────────────────┐
│ 🛒 CARRITO                                       │
├──────────────────────────────────────────────────┤
│ Paracetamol 500mg                                │
│ S/. 9.00 x 2                                     │
│                                                  │
│ [Blister (10 un.) ▼]  [2 ]  S/. 18.00  [🗑️]    │
│                                                  │
│ ------------------------------------------------ │
│                                                  │
│ Ibuprofeno 400mg                                 │
│ S/. 0.80 x 5                                     │
│                                                  │
│ [Unidad ▼]  [5 ]  S/. 4.00  [🗑️]                │
│                                                  │
│ ================================================ │
│ Subtotal:                           S/. 22.00    │
│ Descuento:                         - S/. 0.00    │
│ ------------------------------------------------ │
│ TOTAL:                              S/. 22.00    │
└──────────────────────────────────────────────────┘
```

### 🎮 Cómo funciona:
1. Buscas y agregas producto al carrito
2. Por defecto se agrega como "Unidad"
3. Cambias el selector a "Blister" o "Caja"
4. El precio se actualiza automáticamente
5. Ajustas la cantidad
6. Completas la venta


```
Unidad:  S/. 5.00 (0%)

```

## ❓ PREGUNTAS FRECUENTES

**P: ¿Qué pasa si no ingreso precios por blister/caja?**
R: El sistema los calcula automáticamente con 10% y 20% de descuento.

**P: ¿Puedo cambiar el tipo de venta después de agregar al carrito?**
R: Sí, solo selecciona la nueva opción en el desplegable y el precio se actualiza.

**P: ¿Se puede vender sin descuento?**
R: Sí, ingresa el mismo precio. Ejemplo: Unidad S/. 5.00, Blister S/. 50.00 (0% desc.)

**P: ¿Puedo dar más de 20% de descuento en caja?**
R: Sí, ingresa el precio que quieras. Ejemplo: S/. 70.00 = 30% descuento.

**P: ¿El sistema valida el stock automáticamente?**
R: Sí, no podrás vender más de lo que hay en stock sin importar la presentación.
