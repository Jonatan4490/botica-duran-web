# 💊 Botica Duran - Sistema de Gestión de Farmacia

Sistema completo de gestión para farmacia con control de inventario, ventas, reportes y más.

## 🚀 Características Principales

### 📦 Gestión de Inventario
- Registro de productos por nombre
- Control de stock con alertas de stock mínimo
- Gestión de lotes y fechas de vencimiento
- Categorización de medicamentos
- Alertas de productos por vencer

### 💰 Punto de Venta (POS)
- Interfaz rápida y eficiente para ventas
- Búsqueda rápida de productos
- Múltiples métodos de pago (Efectivo, Tarjeta, Yape, Plin)
- Descuentos y promociones
- Impresión de tickets de venta

### 📊 Reportes y Analytics
- Ventas diarias, semanales y mensuales
- Análisis de ganancias y márgenes
- Productos más vendidos
- Productos por vencer
- Movimientos de inventario
- Historial de ventas

### 👥 Gestión de Clientes
- Registro de clientes
- Historial de compras
- Programa de fidelización

### 📦 Gestión de Proveedores
- Registro de proveedores
- Control de compras
- Historial de precios

### 🔐 Control de Acceso
- Sistema de autenticación
- Perfil de administrador con todos los permisos

## 🛠️ Tecnologías

- **Frontend:** React 18 + PWA (Progressive Web App)
- **Backend:** Node.js + Express
- **Base de Datos:** MySQL
- **Estilos:** CSS Moderno con diseño premium
- **Offline First:** Funciona sin conexión a internet

## 📱 Características PWA

- ✅ Instalable en celular y computadora
- ✅ Funciona offline
- ✅ Sincronización automática cuando hay conexión
- ✅ Notificaciones push
- ✅ Diseño responsive

## 🚀 Instalación

### Requisitos Previos
- Node.js 16 o superior
- MySQL 8.0 o superior

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

## 📝 Configuración

1. Configurar base de datos en `backend/.env`
2. Importar esquema desde `backend/database/schema.sql`
3. Configurar frontend en `frontend/.env`
